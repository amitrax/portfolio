Project Details

Fake News Detector

A Python/Flask web app that uses machine learning to classify news headlines as real or fake. The model was trained on a dataset of labeled news articles and uses NLP techniques for prediction.

Technologies Used

  • Python
  • Flask
  • Scikit-learn
  • NLP
  • HTML/CSS

Features

  • Enter headline to detect real or fake
  • ML Model with accuracy display
  • Clean UI using Tailwind CSS
  • JSON API for prediction endpoint
