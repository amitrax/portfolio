Project Details

Hospital Management System

This web-based application allows patients to book services from different hospital departments such as Pathology, Radiology, and Emergency. Built with Java (Swing UI), MySQL backend, and Java Servlet APIs for backend interaction.

Technologies Used

  • Java (Swing UI)
  • MySQL
  • JDBC & Servlet
  • HTML/CSS

Features

  • Department-wise booking system
  • User login/signup
  • Admin dashboard for managing records
  • Reports and status tracking
